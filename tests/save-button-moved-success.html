📍 Button Location:
❌ Before:

✅ After:
Both buttons are now on the main page

🎯 User Experience:
+ = Immediate action choice
💾 Save to Toolbox Flow:
1. User clicks "🧰 Save to Toolbox" button
2. Simple prompt appears asking for project name
3. Saves directly (localStorage for guests, API for logged-in users)
4. No dialog needed - instant feedback!
🏪 Add to App Store Flow:
1. User clicks "🏪 Add to App Store" button
2. For guests: Shows auth modal to log in first
3. For logged-in users: Opens dialog for project details
4. Publishes to public App Store
